Links And Files
  • Latest Release as a Fullpack The latest full version released as a '.zip' can be found at the Bots-United Filebase under the POD-Bot MM category in Latest release sub-category.
    It contains everything You need to play podbot including *.dll, *.so, waypoints, configs and botchats files. You just need to unzip it in "addons" folder and point the metamod to load it (see installation chapter).
  • Latest Beta .dll The latest windows beta version can be found here. It is work in progress and not recommended to be used in general unless there was a serious problem with the latest release or something important was fixed.
  • Latest Beta .so The latest linux beta version can be found here.
    The latest linux_amd64 beta version can be found here.

    The same as above - it is work in progress and not recommended to be used in general unless there was a serious problem with the latest release or something important was fixed.

  • All-In-One presetup pack The All-In-One installer contains - metamod, AMX MOD X (some latest version) and PODBOT mm. If You don't have idea how to install metamod, podbot and fun modes - use it. Just unzip everything into Your ..\cstrike (or ..\czero) directory and let overwrite all Your existing files. It works for windows and linux (but only 32bit linux).
    You can take it here.
  • Steam Client Steam Clients can be found here, current recommendation by steam to use is the 'steam_installer.exe' (~600kb)
    http://www.steampowered.com/index.php?area=getsteamnow
  • Metamod Metamod is a required plugin for POD-Bot MM to be able to work. It's a mod, but not your ordinary mod, this mod works between the Half-life game and any plugin like POD-Bot MM or AMX MOD X. You can have as many plugins as you want, you can even run multiple bots like YapB and POD-Bot MM in the same game or make all players bleed alot if shot so you can track them down

    Some Plugins Available

    Metamod's Home is http://www.metamod.org
  • POD-Bot MM Waypoints The waypoints for POD-Bot MM can be found almost anywhere so if you have a map with no waypoints then why not try a google search for them. POD-Bot MM can use any of the old POD-Bot v1.0 to v2.5 waypoints as well as the clones of POD-Bot (IvpBot, YaPBBot, E[Pod]Bot)

    Note:

    The only thing that you may find wrong with non-PODBot MM waypoints is that the bots will camp funny, this is because a correction was made to the POD-Bot MM - they were using camp vectors instead of angles. A simple waypoint edit if camp's can be done to fix this, Please check here for how.
